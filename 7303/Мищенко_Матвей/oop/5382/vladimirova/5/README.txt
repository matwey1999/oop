Немодифицирующий лагоритм: Проверка двух неупорядоченных диапазонов на равенство по определенному критерию.
Модифицирующий алгоритм: Удалить элементы, удовлетворяющие заданному критерию.

В качестве немодифицирующего алгоритма используется библиотечная функция std::equal. Для тестирования создаются 2 контейнера
из 1000 фигур. В качестве бинарного предиката используется функция из индивидуального задания isIntersect(), определяющая,
перескаются ли 2 фигуры. Для случая, когда содержимое контейнеров НЕ равно согласно предикату, фигуры обоих контейнеров 
генерируются случайным образом с центром в диапазоне [-diapazon, diapazon] по осям OX и OY и длинами сторон\полуосей в диапазоне [0, diapazon].
Для случая, когда содержимое контейнеров равно согласно предикату, первый контейнер формируется из квадратов с центром в точке (0, 0) и длиной
сторон равной 10, второй контейнер формируется из случайно сгенерированных фигур с центром в диапазоне [-0.5 * diapazon, 0.5 * diapazon] по 
осям OX и OY и длинами сторон\полуосей в диапазоне [diapazon, 2 * diapazon], что гарантирует пересечение любой из сгенерированных фигур из 
второго контейнера с квадратами из первого контейнера.

В качестве модифицирующего алгоритма используется библиотечная функция std::remove_if. Для тестирования создается контейнер
из 1000 фигур. В качестве унарного предиката используется функция, определяющая пересечение с заданной эталонной фигурой при помощи
функции из индивидуального задания isIntersect().
